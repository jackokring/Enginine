# Enginine JUCE Plugin

So with my `VCVRack > KRTPluginA` plugins doing over 15,000+ downloads, I thought I'd give JUCE a go. So this is generic code (2026-05-22)
which I will adapt as I put together some VST3 plugins. It's designed to be edited by `KDevelop` and there are no errors
commited (change the `Parser > Scope: Project` in the tab). Still haven't managed an auto format to `K&R` style yet.
